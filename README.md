# Crowd-Funding

A simple crowdfunding web platform where users can browse, create, and manage fundraising campaigns.

🔗 **Live Demo:** [crowdfundingap.netlify.app](https://crowdfundingap.netlify.app/)

## Features
- Landing page with campaign highlights
- Browse all campaigns (`campaigns.html`)
- Create a new campaign (`create-campaign.html`)
- User dashboard (`dashboard.html`)
- Login page (`login.html`)

## Tech Stack
- HTML
- CSS
- JavaScript (incl. Three.js background)

## Getting Started
```bash
git clone https://github.com/Krishna19-dev/Crowd-Funding.git
cd Crowd-Funding
```
Open `index.html` in your browser to run locally.
